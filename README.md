#  Ομαδικο Project για το μαθημα Αναπτυξης Λογισμικου
Ο στοχος της εργασιας ειναι η επεξεργασια δεδομενων σχετικα με φυσικες καταστροφες για διαφορες χωρες και χρονικες περιοδους. Ο χρηστης μπορει, μεσα απο ενα γραφικο περιβαλλον, να ζητησει πληροφοριες σχετικα με μια φυσικη καταστροφη για μια χωρα που τον ενδιαφερει και να παρει πισω διαφορα γραφηματα.
